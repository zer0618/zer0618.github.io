九妹就是九妹 既不伟大也不卑微——新入N9各种顺手

Posted on 2018-10-07 | In linux

九妹这货就是一个linux移动设备 当初被她可怜没人要的病娇萌到 入手发觉各种好玩啊

首先第一件事当然是同步，旧手机也是诺基亚所以可以很快同步通讯录啥的，然后google的帐号也可以方便导入，联系人之类的搞定。

然后就激活开发者模式， 主要参考这篇 http://www.cuteqt.com/blog/?p=2169

设置->安全性->开发人员模式，拨到右边，系统自动安装Terminal和SDK

ROOT权限的指令是devel-su 密码rootme（好可爱）

ssh登录（我的一直超时不知怎么回事 再弄弄）

$ ssh developer@192.168.2.15 #密码每次自动生成 不想每次输密码的话就把本机公钥拷到N9上

$ ssh-copy-id developer@192.168.2.15 #若提示本地公钥不存在 先执行$ssh-keygen

ssh登录到N9后在$devel-su获得root权限 在键盘操作蛋疼的手机Terminal 其实因为有swype 手机输入还是比想象中快一些的 凹凸的我第一次用swype被惊艳到了

安装软件

N9不像传言中那么缺乏软件支持，有了root权限，各种.deb的包都可以安，我第一个就安了非商店的opera，果然手机上还是opera最习惯啊。但是meego的平板貌似是可以安.rpm的，据说因为英特尔不爽诺基亚吧maemo的东西移植到meego上，所以N9不支持.rpm

方法很简单 把.deb文件扔到Downloads里

$devel-su

$rootme

$cd /home/user/MyDocs/Downloads

$dpkg -i opera.deb #注意文件名要全英文

卸载国行里的鸡肋软件也很简单，$apt-get remove ××××就可以了

安装国行残缺的twitter和gtalk

$cd /etc/apt/sources.list.d/ #修改软件源 你懂的 所以说linux各种顺手唉

$vi aegis.ssu-keyring-003.list #003就是国行的编号

里面的内容如下：

deb https://xxxxxxx:XXXXXXX@downloads.maemo.nokia.com/harmattan/003 ./

deb https://xxxxxxx:XXXXXXX@downloads.maemo.nokia.com/harmattan/apps ./

deb https://xxxxxxx:XXXXXXX@downloads.maemo.nokia.com/harmattan/tools ./

其中每一行对应一个源，从http://到@之间是帐号和密码，应该每台机都不一样吧。文件名以及第一行里的003代表这是国行的源，国际版（还是欧版?）的源是001，那么就把国际版的源添加进去。直接$i把3改成1，:wq，搞定。

然后就是熟悉的$apt-get update #好久没有用过了现在都是pacman 不过可爱的小ubuntu我还是很想你的

之后就可以安装skype呀gtalk呀twitter呀facebook呀啥的：

安装google talk

apt-get install account-plugin-google-talk l10n-app-google

安装skype

apt-get install account-plugin-skype l10n-app-skype

安装twitter

apt-get install twitter l10n-app-twitter webupload-service-twitter

安装youtube

apt-get install youtube-plugin-meta l10n-app-youtube webupload-service-youtube

安装flickr

apt-get install account-plugin-flickr l10n-app-flickr webupload-service-flickr

安装facebook

apt-get install facebook l10n-app-facebook webupload-service-facebook

安装picasa

apt-get install account-plugin-picasa l10n-app-picasa webupload-service-picasa

打开’帐户’，原来的google帐户里会多出一个gtalk的开关

修改hosts

不用迟疑 把电脑里的hosts拷到N9，然后直接$cp /…/hosts /etc/hosts #搞定

顺便记一下vi的命令 如果想要在某个不用的hosts上面加# 可以使用块操作：

光标移到开始位置 比如0移到行头；然后在visual模式（是这个名字吧= =）下进行块操作C-v；光标移到需修改的末尾，比如10行吧，10j；I（大写，为啥大写？我也不知道 我看攻略上这么说orz）插入；插入#；然后按[ESC]退出使其生效。连起来就是：0(C-v)10jI#[ESC]

最后赞扬一下：滑动操作体验很好，整体非常流畅，流畅这一点我想安卓恐怕比较难做到吧，“作恶”的后果。UI设计我在最初看了国行的那个程序图标屏时觉得丑暴了，反复衡量咬牙入手后才发现不是九妹的问题，是那些国内应用的图标实在太丑了，换血以后看着挺舒服的，设置等UI也很简洁干净。还有就是Uni Message概念，据说是从Maemo延续下来的，貌似比imessage和webbos都要早，所有事件放在一起很方便，通讯录也是共用不用一个服务开一个程序很烦，会让人觉得“早就该这样为什么会有手机不是”。
